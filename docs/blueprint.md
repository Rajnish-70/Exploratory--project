# **App Name**: ShikshaPoshan

## Core Features:

- User Account Management: Allow users to securely sign up, log in, and manage their profiles with distinct roles (Administrator, Principal, Teacher, Staff).
- Role-Specific Dashboards: Provide tailored dashboards for each user role, displaying relevant summaries, metrics, and navigation options.
- School & Administrator Management: Enable administrators to add, view, edit school details, manage user accounts, and assign roles within the system.
- Student Academic & Attendance Tracking: Teachers can input and track student attendance and marks, and view detailed student profiles.
- Operational Data Entry (MDM, Security, Cleaning): Staff can input daily data for Mid-Day Meals, security incidents, and cleaning schedules; this data is viewable by relevant roles.
- System-Wide Announcements & News: Administrators can create and broadcast important alerts and news articles visible across all or selected user dashboards.
- Academic Performance Insights Tool: An AI tool that analyzes student marks and attendance trends, identifying potential academic risks and suggesting general pedagogical improvements for education quality.

## Style Guidelines:

- Primary color: A calm, professional medium blue (#336699) to convey trust and education-focused values.
- Background color: A very light, subtle blue-tinted white (#E1EDF2) to ensure readability and an airy feel in a light color scheme.
- Accent color: A deep, growth-oriented green (#29993D) for calls to action and important indicators, providing clear contrast with the primary blue.
- Headline font: 'PT Sans' (humanist sans-serif) for clear and approachable titles across all dashboards.
- Body font: 'Inter' (grotesque-style sans-serif) for optimal readability in data tables, forms, and general text.
- Use a consistent set of clean, modern, outlined icons for navigation, dashboard elements, and form controls to ensure clarity and intuitive interaction.
- Implement structured, responsive grid-based layouts for dashboards and data displays, with clear segmentation for forms and input fields to maintain simplicity and organization.
- Incorporate subtle and swift animation cues for form submissions, navigation transitions, and data loading states to enhance user feedback without being distracting.
